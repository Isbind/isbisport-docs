Installation et Configuration

1
Téléchargement du logiciel

Téléchargez la dernière version du logiciel depuis le lien fourni par l'équipe technique.

Lien de téléchargement : Télécharger ISBISPORT CLUB
2
Installation sur Windows
  1. Double-cliquez sur le fichier d'installation ISBISPORT_Setup.exe
  2. Suivez les instructions à l'écran
  3. Acceptez les termes de la licence
  4. Choisissez le dossier d'installation
  5. Cliquez sur "Installer"
3
Configuration initiale

Lors du premier lancement, vous devrez configurer les paramètres de base :

  • Nom de votre salle de sport
  • Adresse et coordonnées
  • Paramètres de facturation
  • Compte administrateur
Important

Assurez-vous que votre ordinateur répond aux exigences système minimales avant l'installation :

  • Windows 10 ou version ultérieure
  • 4 Go de RAM (8 Go recommandés)
  • 500 Mo d'espace disque disponible
  • Connexion Internet pour l'activation

Gestion des Membres

Ajouter un nouveau membre

1

Dans le menu principal, cliquez sur Membres puis sur Nouveau Membre.

2

Remplissez le formulaire avec les informations du membre :

  • Informations personnelles (nom, prénom, date de naissance, etc.)
  • Coordonnées (adresse, téléphone, email)
  • Photo (optionnelle)
  • Informations médicales (optionnel)

Rechercher un membre

Utilisez la barre de recherche en haut de l'écran pour trouver rapidement un membre par son nom, prénom ou numéro d'identification.

Astuce

Vous pouvez utiliser des filtres avancés pour afficher les membres selon différents critères (abonnements actifs, anniversaires du mois, etc.).

Gestion des Abonnements

Types d'abonnements

Le système prend en charge différents types d'abonnements :

Type Durée Prix Accès
Mensuel 1 mois 15,000 FCFA Accès illimité
Trimestriel 3 mois 40,000 FCFA Accès illimité
Annuel 12 mois 120,000 FCFA Accès illimité + 1 mois offert
Séance 1 jour 1,500 FCFA Accès 1 jour

Créer un nouvel abonnement

1

Sélectionnez un membre depuis la liste ou créez-en un nouveau.

2

Cliquez sur Nouvel Abonnement et sélectionnez le type d'abonnement.

3

Vérifiez les détails et confirmez la création de l'abonnement.

Important

N'oubliez pas d'enregistrer le paiement associé à l'abonnement pour l'activer. Les abonnements non payés seront marqués comme "En attente".

Gestion des Paiements

Enregistrer un paiement

1

Depuis le profil d'un membre, cliquez sur Nouveau Paiement.

2

Sélectionnez le type de paiement :

  • Abonnement
  • Séance ponctuelle
  • Produit
  • Autre
3

Remplissez les détails du paiement :

  • Montant
  • Mode de paiement (Espèces, Carte, Virement, etc.)
  • Date du paiement
  • Notes (optionnel)

Facturation

Le système génère automatiquement une facture pour chaque paiement. Vous pouvez :

  • Imprimer la facture
  • Envoyer la facture par email
  • Télécharger la facture au format PDF
Rappels automatiques

Le système peut envoyer automatiquement des rappels par SMS ou email pour les paiements en retard. Configurez cette fonctionnalité dans les paramètres de notification.

Gestion des Présences

Enregistrer une entrée

1

Dans l'écran d'accueil, utilisez le lecteur de code-barres ou recherchez manuellement le membre.

2

Le système affiche la photo et les informations du membre. Vérifiez que l'abonnement est valide.

3

Cliquez sur Enregistrer l'entrée. L'heure d'entrée est automatiquement enregistrée.

Enregistrer une sortie

Lorsque le membre quitte la salle, recherchez-le à nouveau et cliquez sur Enregistrer la sortie.

Important

Si un membre oublie d'enregistrer sa sortie, le système peut être configuré pour l'enregistrer automatiquement après une certaine période d'inactivité.

Planification des rapports

Planifiez l'envoi automatique de rapports par email :

  1. Allez dans Paramètres > Rapports automatisés
  2. Cliquez sur Nouveau rapport automatisé
  3. Sélectionnez le type de rapport et la fréquence
  4. Ajoutez les destinataires
  5. Enregistrez la configuration

Maintenance et Support

Sauvegarde des données

Il est essentiel d'effectuer régulièrement des sauvegardes de vos données :

1

Dans le menu principal, allez dans Paramètres > Sauvegarde et Restauration.

2

Cliquez sur Créer une sauvegarde.

3

Choisissez un emplacement de stockage sécurisé pour votre fichier de sauvegarde.

Conseil de sécurité

Stockez toujours vos sauvegardes dans un emplacement différent de votre ordinateur principal. Utilisez un disque dur externe ou un service de stockage en cloud sécurisé.

Mises à jour logicielles

Assurez-vous d'utiliser toujours la dernière version du logiciel :

  • Les mises à jour incluent des corrections de bugs et de nouvelles fonctionnalités
  • Vérifiez régulièrement les mises à jour dans Aide > Rechercher les mises à jour
  • Créez toujours une sauvegarde avant d'installer une mise à jour

Foire Aux Questions

Pour ajouter un nouveau type d'abonnement :

  1. Allez dans Paramètres > Types d'abonnements
  2. Cliquez sur Nouvel abonnement
  3. Remplissez les détails (nom, durée, prix, avantages)
  4. Enregistrez les modifications

Pour créer un rapport personnalisé :

  1. Allez dans Rapports > Nouveau rapport
  2. Sélectionnez le type de données à inclure
  3. Appliquez des filtres si nécessaire
  4. Choisissez le format de sortie (PDF, Excel, etc.)
  5. Générez et enregistrez le rapport

En cas de problème technique :

  1. Redémarrez l'application
  2. Vérifiez votre connexion Internet
  3. Assurez-vous d'utiliser la dernière version du logiciel
  4. Consultez la section d'aide en ligne
  5. Si le problème persiste, contactez le support technique
